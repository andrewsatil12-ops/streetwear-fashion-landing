2. Built a layout that prioritizes imagery, rhythm, and visual impact  
3. Structured sections to balance brand storytelling and product perception  
4. Refined typography, spacing, and contrast to reinforce authenticity and attitude  

## Result
A bold and modern landing page that communicates streetwear identity while maintaining clarity, usability, and brand consistency.
